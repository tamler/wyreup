Add eight ffmpeg.wasm media tools: resize-video, mute-video, rotate-video, extract-frame, replace-audio, normalize-loudness, analyze-loudness, and video-quality-metrics (PSNR/SSIM). All run client-side, free tier, on the existing ffmpeg core — no new dependencies. Loudness targets cover EBU R128, ATSC A/85, Spotify, Apple Music, YouTube, and Amazon.
